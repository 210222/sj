方案对比：JSXGraph vs Plotly vs ECharts vs Mermaid
JSXGraph ✅ AI 可生成 (JSON) ✅ npm install 离线 ✅ 交互式拖动/旋转 ✅ 几何 + 代数 + 函数 ✅ 可出考试题 ⚠️ 文档偏学术 适合：数学、几何、物理
Plotly.js ✅ AI 可生成 (JSON) ✅ npm install 离线 ✅ 3D 曲面/散点/线 ✅ 交互式旋转/缩放 ❌ 不能画几何图形 适合：3D 数学、数据可视化
ECharts ✅ AI 可生成 (JSON) ✅ npm install 离线 ✅ 图表类型最多 ❌ 不能交互拖动 ❌ 不能画几何图 适合：统计、对比、树形
Mermaid
✅ 文本→图形
✅ 流程图/时序/思维导图
❌ 不能画函数/几何
适合：概念、流程、关系
KaTeX
✅ LaTeX→公式
✅ 极轻量
❌ 纯文本渲染
适合：数学公式
推荐组合
