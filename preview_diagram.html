场景 2：讲解三角函数 sin(x) — 数学公式渲染
sin 函数的周期是什么意思？
周期就是"重复一次需要走多远"。sin(x) 的周期是，意味着每隔 2π，图像完全重复：
📊 概念关系图
graph LR
    A["输入角度 x"] --> B["sin(x) 计算"]
    B --> C["输出值 y ∈ [-1, 1]"]
    C --> D["每 2π 重复"]
    D --> A
你可以这样理解：就像一个钟摆，从中间→最右→中间→最左→中间，一个完整的来回就是一圈（2π）。你用一句话说说 sin(π/2) 应该是多少？
场景 3：讲解冒泡排序 — 步骤流程图
冒泡排序是怎么一步步排的？
冒泡排序就像气泡从水底往上升——每次把最大的元素"浮"到最后面：
📊 步骤流程图
flowchart TD
    A["[5, 3, 8, 1]"] --> B["第一轮: 5↔3 交换"]
    B --> C["[3, 5, 8, 1] → 5↔8 不动"]
    C --> D["[3, 5, 8, 1] → 8↔1 交换"]
    D --> E["第一轮结束: [3, 5, 1, 8]"]
    E --> F["第二轮: 把 5 浮到最后"]
    F --> G["...逐轮重复直到有序"]
关键规律： 现在给你 [4, 2, 7, 1]，第一轮结束后会是什么样？自己推一下。
场景 4：三角函数 sin(x) — 函数曲线图 (XY Chart)
sin(x) 的图像长什么样子？
你看，这就是 sin(x) 在 0 到 2π 之间的图像——想象一个弹珠在圆周上滚动的投影高度：
📊 sin(x) 函数图像 [0, 2π]
xychart-beta
    title "正弦函数 y = sin(x)"
    x-axis "角度" [0, "π/2", "π", "3π/2", "2π"]
    y-axis "y 值" -1 --> 1
    line "sin(x)" [0, 1, 0, -1, 0]
你看：——从 0 开始升到最高点 (π/2)，再降到 0 (π)，再降到最低点 (3π/2)，最后回到 0 (2π)。这个波浪就是 sin 的"周期"。现在你想象一下 cos(x) 会从哪个点开始？
场景 5：HTTP 请求-响应 — 时序图
浏览器输入网址后发生了什么？
从你敲回车到页面出现，经历了这些步骤：
📊 HTTP 请求时序图
sequenceDiagram
    participant 浏览器
    participant DNS

    浏览器->>DNS: 1. "把 example.com 翻译成 IP 地址"

    浏览器->>服务器: 3. "GET /index.html"
