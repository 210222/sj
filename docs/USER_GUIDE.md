# Coherence Coach 用户指南

Coherence Coach 是一个 **AI 学习教练**。它会根据你的学习阶段、动机状态和能力水平，动态调整教学内容——不是给你扔一堆资料，而是像一个真正的教练，在你需要的时候给你恰好需要的帮助。

---

## 1. 启动

### 第一步：启动后端

打开一个终端（cmd 或 PowerShell）：

```cmd
set DEEPSEEK_API_KEY=YOUR_DEEPSEEK_API_KEY
cd D:\Claudedaoy\coherence
python -m uvicorn api.main:app --host 127.0.0.1 --port 8001
```

看到 `Application startup complete` 就说明后端跑起来了。

### 第二步：启动前端

另开一个终端：

```cmd
cd D:\Claudedaoy\coherence\frontend
npm run dev
```

浏览器打开它显示的地址（默认 **http://localhost:5173**）。

### 移动端（可选）

```cmd
cd D:\Claudedaoy\coherence\mobile
flutter run -d edge
```

## 2. 基本使用

在网页的对话框里输入你想学的内容，教练会回复你。例如：

| 你说 | 教练会做的事 |
|------|------------|
| "教我 Python 列表" | 生成一份结构化的 Python 列表教程 |
| "为什么 for 循环比 while 快" | 解释底层原理，引导你思考 |
| "考考我排序算法" | 出题检测你的掌握程度 |
| "给我一个有点难的挑战" | 生成略高于你当前水平的任务 |
| "我想暂停一下" | 退一步，不继续推进 |

## 3. 四种学习模式

### 3.1 基础模式（默认）

LLM 生成内容，使用基础教练策略。适合日常学习。

### 3.2 行为科学增强模式

在 `config/coach_defaults.yaml` 中打开三个开关：

```yaml
ttm:
  enabled: true      # 学习阶段检测
sdt:
  enabled: true      # 动机评估
flow:
  enabled: true      # 难度动态调节
```

重启后端后生效。效果：

- **TTM（阶段检测）**：系统判断你当前的学习状态——还没开始？正在犹豫？已经行动？正在坚持？不同阶段给不同类型的引导。
- **SDT（动机评估）**：检测你的自主性、胜任感和关联性。如果你频繁改写建议 = 高自主性 → 给你更多选择。如果你很少改写 = 低自主性 → 引导你表达自己的判断。
- **心流调节**：你在无聊和焦虑之间来回时，动态调节难度。做得好 → 加难度。表现挣扎 → 降难度。

### 3.3 探索模式

在对话框输入 `/excursion` 进入探索模式。界面切换为深色主题，系统暂时忽略你的学习历史，给你完全自由的思考空间。适合发散联想。

### 3.4 流式输出模式

在配置中开启 `llm.streaming: true`，AI 回复会像打字一样逐字显示，更有对话感。

## 4. 你的权利

### 4.1 主权脉冲

当系统准备给你一条高影响力的建议时，会先弹出一个**确认面板**：
- *滑动接受*：同意系统的建议
- *改写前提*：用自己的话重新定义问题的前提

你有权拒绝、改写或退出任何教练建议。

### 4.2 认知主权声明

系统会周期性提醒你：*"你可以拒绝、改写或退出当前画像——这是你的认知主权。"*

### 4.3 隐私

- 你的学习数据存储在本地 SQLite 文件中，不上传任何服务器
- API Key 只存在于环境变量中，不写入文件、不记录日志
- 所有安全门禁在本地执行

## 5. 管理后台

在设置页面切换角色为"管理员"，可以查看：

- **8 道安全门禁**：每条回复都经过 8 项安全检查（话语支配风险 / 探索深度 / 独立思考轨迹 / 顺从信号 / 因果推断 / 审计健康 / 框架思维 / 版本一致性）
- **审计日志**：按严重级别筛选历史事件
- **风险评估**：已知系统风险及建议

## 6. 常见问题

### 为什么回复很短/像是模板？

LLM 模式未开启。检查 `coach_defaults.yaml` 中 `llm.enabled: true`，确保启动后端前设置了 `DEEPSEEK_API_KEY`。

### 为什么回复是乱码？

Windows 终端的编码问题。在浏览器里看回复是正常的。

### 怎么能让教练更了解我的水平？

打开行为科学模型（TTM + SDT + 心流），多和教练对话。系统会根据你的交互历史逐步调整。

### 移动端和网页端有什么区别？

功能完全一致。移动端适配了小屏交互，支持滑动确认和底部导航。

### 系统安全吗？

每条 AI 回复都经过 **8 道门禁** 安全校验。如果 AI 生成的内容被判定为不安全，会自动替换为安全版本。代码执行在沙箱中隔离运行（禁止系统调用/网络操作），超时 10 秒自动终止。

## 8. 配置参考

所有可调参数在 `config/coach_defaults.yaml`：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `llm.enabled` | false | 是否启用 AI 教练 |
| `llm.model` | deepseek-chat | AI 模型 |
| `llm.temperature` | 0.7 | 创造性（0=保守，1=天马行空） |
| `llm.max_tokens` | 2000 | 单次回复最大长度 |
| `llm.streaming` | false | 是否逐字输出 |
| `ttm.enabled` | false | 学习阶段检测 |
| `sdt.enabled` | false | 动机状态评估 |
| `flow.enabled` | false | 难度动态调节 |
